# Traffic Violation Detection

A computer vision-based system for detecting traffic violations using YOLOv8.

## Features

* Helmet detection
* Number plate detection
* Traffic violation monitoring
* Web-based user interface

## Tech Stack

* Python
* Flask
* YOLOv8
* HTML
* CSS
* JavaScript

## Project Structure

* app.py - Detection pipeline
* Weights/ - YOLO model weights
* Test Images/ - Sample test images
* index.html - Frontend
* style.css - UI styling
* script.js - API integration

## Installation

```bash
pip install -r requirements.txt
python app.py
```
